TMU Payment Popup - Standalone JavaScript Library

This is a complete, standalone TMU payment popup that can be integrated into any website without any dependencies.

Basic Usage

<!-- Include the TMU Payment script --> <script src="tmuPayment.js"></script> <script> // Basic usage TMUPayment.open({ amount: 99.99, onSuccess: (result) => { alert('Payment successful!'); console.log(result); }, onCancel: () => { console.log('Payment cancelled'); } }); </script>

Advanced Usage

TMUPayment.open({ amount: 249.50, currency: 'USD', onSuccess: (result) => { console.log('Transaction ID:', result.transactionId); // Redirect to success page window.location.href = '/success'; }, onCancel: () => { console.log('User cancelled payment'); }, onError: (error) => { console.error('Payment failed:', error); alert('Payment failed. Please try again.'); } });
